Display your active listings from myRealPage on your Wordpress blog

== Description ==

This widget is for those who blog using Wordpress and have a website for their listings at myRealPage. You want to be able to list your listings on the sidebar of your blog, now you can.

== Installation ==

1. Extract and upload 'mrp-listings' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit your theme widgets and drag the new myRealPage widget to the sidebar
4. Configure your title and domain name (remember that you need to use your exact domain name)

== Frequently Asked Questions ==

= I am getting errors when saving my details =

Have you tried removing the www or adding the www to your domain name in the widget settings?
